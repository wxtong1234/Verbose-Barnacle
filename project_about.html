What is Our Project

    Many photographer use different app to help them take batter picture. For example, Astorphotographer will install an Celestial event reminder app for them to retrive information easily. photographer with many gear will install an gear management app for them to keep track of those grears. Natural photographer will install an shutter speed calculator for long exposer shoot. All these app are very helpful for them. But this usually result installing multiple apps. Our app conbind all these feature into one app. By doing so reduce the frastration on opening multiple apps.

Our Goal

    Our project is a photo application that help photographer both professional and Ameture takes batter photo. The app will run on Andorid Sysyem. It is design to be an all in one tool box for photographer. The App inclued many useful feature such as user login, Gear Management,Real Time GPS Record, Camera Caclualtor, Weather Information,EXIF Information Editor and Celestial Event. With those tools user can takes batter picture.

Feature

    Login

        user login is useful when you transfer your account to another phone. THe Gear Management and GPS information will be avaliable when you switch your phone.

    Gear Management

        Gear Management is expecially useful for photographers who has a lot of Gears. You can write the status of the Gear such as broken, borrow, sent for fix. You can also input the insurence and worrenty due date so you will never forget the date. You can also take not on each gear such as best aperture, best focal length and so on.

    Real Time GPS Record

        GPS location infromation is useful in many situation. The photographer can sync the GPS location. with the EXIF information. User can download the GPS location information from their phone or via email. The GPS information will be transfer withe account if the user switch phone.

    Camera Caclualtor

        The Camera Calculator is useful for calculating the apeture, shutter speed, ISO and ND filter. By entering three out of four setting into the calcualtor. calcualtor will give the result of the disire setting.

    Weather Information

        Weather information is useful during prepping. User can check the weather base on the location or entering the location name

    EXIF Information Editor

        EXIF information editor let photographers to edit the EXIF information on phone. It is good idea to remove the EXIF informati before you post the photo on soicial media.

    Celestial Event

        Celstial Event let user check list of Clestial Event and Star the interest event

What Makes Our Project Unique

    Differ form other app. Our App focus on bring all the tool into one app. Also we provied many other usefull feature such as GPS tracking so that location information can be tag on EXIF informatin during postprocessing.
